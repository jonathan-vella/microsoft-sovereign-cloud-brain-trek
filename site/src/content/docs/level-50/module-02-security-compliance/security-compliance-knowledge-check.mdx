## Questions

### Question 1

What does the CIA triad in information security represent?

**A)** Central Intelligence Agency guidelines
**B)** Confidentiality, Integrity, Availability
**C)** Cloud Infrastructure Architecture
**D)** Compliance, Identity, Access

<KnowledgeCheck answer="B" reference="/level-50/module-02-security-compliance/security-compliance-basics/">

**Explanation:** The CIA triad represents the three fundamental principles of information security.

</KnowledgeCheck>

### Question 2

Which authentication factor includes passwords and PINs?

**A)** Something you are
**B)** Something you have
**C)** Something you know
**D)** Somewhere you are

<KnowledgeCheck answer="C" reference="/level-50/module-02-security-compliance/data-protection-principles/">

**Explanation:** Knowledge factors include passwords, PINs, and security questions.

</KnowledgeCheck>

### Question 3

What is the principle of least privilege?

**A)** Giving users maximum access for convenience
**B)** Providing minimum necessary permissions for job function
**C)** Allowing equal access for all users
**D)** Restricting access only to sensitive data

<KnowledgeCheck answer="B" reference="/level-50/module-02-security-compliance/compliance-frameworks/">

**Explanation:** Least privilege means providing only the minimum permissions necessary.

</KnowledgeCheck>

### Question 4

</KnowledgeCheck>

### Question 5

### Question 8

### Question 9
